If you discover a security vulnerability, please do **not** report it publicly. Email us at [security@appfoster.com](mailto:security@appfoster.com) for a coordinated disclosure.

## Issue Reporting Tips
When contacting support or opening a GitHub issue, providing the following information will help us resolve your problem faster:

1.  **Plugin Version:** (e.g., 1.0.0)
2.  **WordPress Version:** (e.g., 6.4.2)
3.  **PHP Version:** (e.g., 8.1.0)
4.  **Error Messages:** Copy and paste the exact text of any error messages you see.
5.  **Screenshots:** Attach images of the issue if applicable.
6.  **Steps to Reproduce:** A brief list of what you were doing when the problem occurred.
