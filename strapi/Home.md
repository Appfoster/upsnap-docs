# Welcome to the UpSnap Monitoring Documentation

Welcome to the official documentation for the **UpSnap Monitoring** plugin for Strapi.

## Overview
UpSnap is a lightweight, all-in-one monitoring plugin for **Strapi v5** that provides real-time insights into your website's health, performance, and security directly from your Strapi admin panel.
