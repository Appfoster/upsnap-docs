## 🛠️ Core Features

Across all platforms, UpSnap delivers a powerful suite of monitoring tools:

-   **Global Reachability:** Real-time uptime tracking from multiple global regions.
-   **SSL/TLS Security:** Automated monitoring of certificate expiry and chain validity.
-   **Lighthouse Performance:** Integration with Google Lighthouse for Performance, SEO, and Accessibility audits.
-   **Broken Link Detection:** Proactive scanning to identify and fix dead links.
-   **Mixed Content Alerts:** Security scanning to detect insecure resources on HTTPS pages.
-   **Domain Health:** Tracking of DNS records and domain registration status.
